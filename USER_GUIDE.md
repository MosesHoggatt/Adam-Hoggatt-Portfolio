# Quick Start Guide for Non-Technical Users

This guide will help you upload and manage projects on your portfolio website.

## Accessing the Admin Dashboard

1. Open your portfolio website
2. Add `/admin` to the end of the URL (e.g., `https://yoursite.com/admin`)
3. You'll see a login screen

## First Time Setup - Creating Your Account

1. Click the **"Create Account"** tab
2. Enter your **email address**
3. Create a **password** (must be at least 8 characters)
4. Click **"Create Account"**
5. Check your email inbox (and spam folder) for a verification code
6. Enter the verification code when prompted
7. You're now registered! Sign in with your email and password

## Uploading a New Project

1. Sign in to the admin dashboard
2. You'll see a form titled **"Add New Project"**
3. Fill in the project details:
   - **Project Title**: The name of your project (e.g., "Nuketown")
   - **Description**: A description of the project (what it is, your role, etc.)
   - **Date**: The project date (defaults to today)

4. **Adding Images**:
   - You can either:
     - **Drag and drop** images onto the box that says "Drag & drop images here"
     - **Click** the box to select images from your computer
   - You can select multiple images at once
   - You'll see thumbnail previews of all selected images

5. Click **"Upload Project"**
6. Wait for the success message
7. Your project will appear in the "Existing Projects" list below

## Viewing Your Projects

### On the Public Website
1. Go to your main website URL (without `/admin`)
2. Scroll to the "My Projects" section
3. Your newly uploaded projects will appear automatically
4. No page refresh or rebuild needed!

### In the Admin Dashboard
1. Scroll down to the **"Existing Projects"** section
2. You'll see a list of all your projects with:
   - Project title
   - Description
   - Date
   - Number of images

## Deleting a Project

1. In the admin dashboard, scroll to **"Existing Projects"**
2. Find the project you want to delete
3. Click the red **"Delete"** button next to it
4. Confirm the deletion when asked
5. The project and all its images will be permanently removed

## Signing Out

1. Look at the top right corner of the admin page
2. Click the **"Sign Out"** button
3. You'll be returned to the login screen

## Tips for Best Results

### Images
- Use high-quality images (at least 1920x1080 for best results)
- Supported formats: JPG, PNG, GIF, WebP
- Recommended size: Under 5MB per image
- The first image you upload will be used as the main thumbnail

### Project Titles
- Keep titles clear and concise
- Good examples: "Nuketown", "Express Map Design", "Raid Level Design"

### Descriptions
- Describe your role in the project
- Mention key features or achievements
- Keep it between 50-200 words for readability

### Organization
- Upload projects in chronological order (newest first)
- Add projects consistently to keep your portfolio up to date

## Troubleshooting

### Can't Sign In
- **Check your email and password** - make sure caps lock is off
- **Forgot password?** Click "Forgot your password?" on the login screen
- **Not verified?** Check your email for the verification link

### Images Not Uploading
- **Check file size** - images should be under 10MB each
- **Check file format** - use JPG, PNG, GIF, or WebP
- **Check internet connection** - large uploads need stable internet

### Images Not Showing on Public Site
- **Wait a minute** - sometimes there's a small delay
- **Clear browser cache** - press Ctrl+Shift+R (or Cmd+Shift+R on Mac)
- **Check admin dashboard** - make sure the project shows in "Existing Projects"

### Upload Taking Too Long
- **Large images** take longer to upload
- **Multiple images** will take more time
- Don't close the browser while "Uploading..." is shown
- Wait for the success message before navigating away

## Getting Help

If you encounter issues:
1. Check this guide first
2. Try signing out and signing back in
3. Try a different browser (Chrome, Firefox, Safari, Edge)
4. Contact your web developer with:
   - What you were trying to do
   - What happened instead
   - Any error messages you saw (take a screenshot)

## Security Reminders

- **Never share your password** with anyone
- **Use a strong, unique password** for this account
- **Sign out** when you're done, especially on shared computers
- Only you (and other admins you authorize) can access `/admin`
- The public cannot see the admin dashboard or your login

---

**That's it!** You're ready to manage your portfolio. If you have any questions, reach out to your web developer.
