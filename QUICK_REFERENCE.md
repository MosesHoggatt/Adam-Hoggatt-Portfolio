# Quick Reference Card - Admin Dashboard

## 🔐 How to Access Admin
1. Go to your website URL
2. Add `/admin` at the end
3. Example: `yoursite.com/admin`

## 📝 Upload a New Project

### Required Information:
- ✅ Project Title (e.g., "Nuketown")
- ✅ Description (what it is, your role)
- ✅ At least 1 image
- ✅ Date (optional, defaults to today)

### Steps:
1. Fill in title and description
2. Click the upload box OR drag images onto it
3. See preview thumbnails
4. Click "Upload Project"
5. Wait for "Success!" message
6. Done! Project appears on your site immediately

## 🗑️ Delete a Project

1. Scroll to "Existing Projects"
2. Find the project
3. Click red "Delete" button
4. Confirm deletion
5. Done!

## 💡 Pro Tips

### Images
- Use high quality (1920x1080+)
- Keep under 5MB each
- JPG or PNG work best
- First image = thumbnail

### Titles
- Keep short and clear
- Example: "Raid Map Design"

### Descriptions
- 50-200 words ideal
- Mention your role
- Highlight key features

## 🆘 Quick Fixes

### Can't Login
- Check email/password (caps lock?)
- Click "Forgot password?"
- Check spam for verification email

### Upload Failed
- Check file size (under 10MB)
- Check internet connection
- Try one image at a time
- Refresh page and try again

### Images Not Showing
- Wait 1 minute
- Press Ctrl+Shift+R to refresh
- Check if upload succeeded

## 🔒 Security

- ✅ Always sign out when done
- ✅ Never share your password
- ✅ Use strong, unique password
- ✅ Public cannot access admin
